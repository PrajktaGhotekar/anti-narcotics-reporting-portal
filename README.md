# Anti-Narcotics Reporting & Awareness Portal

A full-stack web application built with React.js and Spring Boot that allows users to report narcotics-related activities anonymously.

## Tech Stack
- Frontend: React.js, Vite, HTML, CSS
- Backend: Spring Boot, Java
- Database: MySQL
- APIs: REST APIs
- Authentication: JWT

## Features
- Anonymous narcotics reporting
- Image upload with reports
- Admin dashboard for case management
- Secure authentication
- Razorpay donation integration
- RESTful backend APIs

## Project Structure
Frontend and Backend code are included in this repository.
